#### 什么编码问题
例如，`工`和`⼯`，第一个就是标准汉字，Unicode为`U+5DE5`，第二个是部首符号，Unicode为`U+2F2E`，大概就是爬虫时字体映射不完全，一种常见的反爬虫机制（？）
